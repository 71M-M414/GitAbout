Comandos do Git

Nesta página, você verá os principais comandos do Git:

  • git init
    para inicializar um repositório;

  • git status
    para obter status do repositório;

  • git add .
    para adicionar arquivos novos e alterados;

  • git commit -m "mensagem de alteração"
    para fazer as mudanças no repositório;

  • git pull origin nome-do-branch
    para ver o histórico do servidor remoto;

  • git push origin nome-do-branch
    para enviar o histórico ao servidor remoto;

  • git branch
    para listar as branches;

  • git checkout nome-do-branch
    para alternar de branch;

  • git branch nome-do-branch
    para criar um branch;

  • git checkout -b new-issue
    para criar e selecionar o branch;

  • git branch -d nome-do-branch
    para deletar um branch local;

  • git push origin :nome-do-branch
    para deletar um branch remoto;
